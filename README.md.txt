# Cartilla web: prevención de violencias basadas en género en las niñeces

Cartilla digital interactiva desarrollada como entregable de tesis de maestría.

## Características

- Diseño web responsive.
- Compatible con computador, tablet y celular.
- Navegación por desplazamiento suave.
- Punto de partida con acordeón interactivo.
- Navegación por ocho ejes pedagógicos.
- Diseño con enfoque de accesibilidad.
- Paleta visual basada en morados y blancos.
- Publicación mediante GitHub Pages.

## Archivos

- index.html → estructura y contenido de la cartilla.
- styles.css → diseño visual.
- script.js → interacciones.

## Cómo publicarla en GitHub Pages

### 1. Crear repositorio

Ingresa a GitHub:

https://github.com

Crea un nuevo repositorio.

Ejemplo:

cartilla-vbg


### 2. Subir archivos

Sube estos tres archivos:

- index.html
- styles.css
- script.js


### 3. Activar GitHub Pages

Dentro del repositorio:

1. Entra a Settings.
2. Busca la opción Pages.
3. En "Branch" selecciona:

main

4. Carpeta:

/root

5. Guarda.

GitHub generará una dirección pública.


### 4. Abrir la página

Después de unos minutos aparecerá:

https://tuusuario.github.io/cartilla-vbg/


## Nota académica

La estructura digital fue diseñada para facilitar la lectura y navegación del documento original sin modificar el contenido académico de la cartilla.
